# LinkedIn Project Description (Professional & AI-Highlighted)

## 🎯 SpendWise – AI-Powered Personal Finance Assistant

Built an intelligent expense tracking application featuring a conversational AI agent that understands natural language commands to manage personal finances effortlessly.

### 🤖 AI Integration Highlights:
- Conversational AI Agent powered by Gemini 2.5 Flash Lite
- Natural Language Processing for expense commands (e.g., "Add coffee 150 PKR in Food category")
- AI-driven expense categorization & insights
- Real-time intent recognition for CRUD operations
- Context-aware responses with structured data extraction
- OpenAI-compatible API integration for seamless LLM interaction

### ✨ Core Features:
- 🔐 Firebase Authentication with Protected Routes
- 💰 Real-time Expense & Income Management
- 📊 Interactive Analytics Dashboard (Chart.js)
- 🗣️ AI Chatbot for Voice/Text Commands
- 🌙 Dark/Light Theme with shadcn/ui Components
- 📱 Fully Responsive Design
- 🌍 Multi-Currency Support

### 🛠️ Tech Stack:
- **Frontend:** Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Firebase (Auth, Firestore, Admin SDK), Next.js API Routes
- **AI/ML:** Gemini 2.5 Flash Lite, OpenAI Agents SDK, Zod Schema Validation
- **State Management:** Zustand
- **Visualization:** Chart.js, React-Chartjs-2
- **Deployment:** Vercel

### 🔗 Live Demo: https://expenses-tracker.vercel.app
